# Ahtisham Irfan — Portfolio Website

A code-editor themed animated portfolio (VS Code style UI) built with vanilla HTML, CSS, and JavaScript.

## Files
- `index.html` — page structure/content
- `style.css` — all styling
- `script.js` — tab switching, typing animation, project commit expand/collapse

## Run locally
Just open `index.html` in a browser — no build step needed.

## Deploy on GitHub Pages
1. Create a new repo named `ahtisham-portfolio.github.io`
2. Upload these 3 files (index.html, style.css, script.js) to the repo root
3. Go to Settings → Pages → confirm it's serving from the `main` branch
4. Your site goes live at `https://github.com/Ahtisham-Irfan`
